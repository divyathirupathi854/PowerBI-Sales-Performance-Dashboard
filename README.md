📊 Power BI Sales Performance Dashboard

📌 Project Overview
          This project is an interactive Sales Performance Dashboard developed using Power BI. It provides valuable business insights by analyzing sales performance, customer behavior, product performance, and profitability through interactive visualizations and KPIs.

📊 Dashboard Highlights

- Sales Overview
- Monthly & Yearly Sales Trends
- Top Customers & Top Products Analysis
- Customer Summary
- Profit & Revenue Analysis
- Segment Analysis
- Profitability Analysis
- Dynamic Filters & Interactive Reports

📈 KPIs Included

- Total Sales
- Total Orders
- Total Quantity Sold
- Total Customers
- Profit
- Profit Margin %
- Sales Growth %
- Returning Customer %

🛠️ Tools & Technologies Used

- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Cleaning
- Data Visualization
